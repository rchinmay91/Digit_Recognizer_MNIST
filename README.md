# 🔢 Digit Recognizer (MNIST) - Google Colab Solution

This repository contains my Google Colab implementation for the Kaggle **Digit Recognizer** competition, using the classic MNIST dataset of handwritten digits.

## 🚀 Project Overview
* **Goal**: Correctly identify handwritten digits (0-9) from an image dataset.
* **Environment**: Developed and trained entirely in Google Colab.
* **Libraries Used**: Pandas, NumPy, Scikit-Learn.

## 📊 Model Performance & Results
* **Algorithm**: Random Forest Classifier
* **Local Validation Accuracy**: **94.31%**
* **Official Kaggle Leaderboard Score**: **0.98546 (98.55% Accuracy)** 🚀🎯
* **Status**: Successfully submitted and verified on Kaggle.

## 📦 How to Use This Project
1. Open the `.ipynb` notebook file in this repository.
2. Click the **"Open in Colab"** badge at the top of the notebook.
3. Download the competition data (`train.csv` and `test.csv`) from [Kaggle](https://kaggle.com).
4. Upload the datasets to your Colab session storage and run all cells to train the model and generate your submission file.
